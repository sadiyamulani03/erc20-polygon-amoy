# ERC-20 Token Deployment on Polygon Amoy 🚀

## 📌 Overview
This project demonstrates deployment of an ERC-20 token on the Polygon Amoy Testnet as part of Blockchain Platform Laboratory.

## ⚙️ Tech Stack:-
- Solidity
- OpenZeppelin
- Remix IDE
- MetaMask
- Chainlist

## 🛠️ Steps Performed:-
1. Added Polygon Amoy network using Chainlist
2. Wrote ERC-20 smart contract using OpenZeppelin
3. Compiled and deployed using Remix IDE
4. Connected MetaMask wallet
5. Executed token transfer transactions
6. Imported the GLD Token in MetaMask

## ⚡ Key Learning:-
- Understanding Layer 2 transaction behavior
- Lower gas fees on Polygon network
